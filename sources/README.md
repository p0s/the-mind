# Sources

Canonical source index:
- `sources/sources.csv`

Guidelines:
- One row per source (YouTube video, podcast episode, essay, talk, etc.).
- Keep `source_id` stable (used to name notes and any local artifacts).
- Prefer timecoded sources when possible (video/podcasts).

Recommended `source_id` schemes:
- YouTube: `yt_<video_id>` (example: `yt_E361FZ_50oo`)
- CCC: `ccc_<slug>`
- Web/articles: `web_<host>_<path>` (generated; keep stable once present)
